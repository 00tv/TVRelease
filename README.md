## 使用说明：

1、下载安装运行，点右下角的设置图标   零零影视—兼容TVBox接口影视APP安卓手机魔改竖版TVBox下载

2、点击如下图”点播“右边空余位置，然后输入兼容TVBox的接口网址，再点确定。

3、OK了，享受免费的VIP视频服务吧。点击首页右上角”换源“可以更换片源线路。
